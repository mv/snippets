To install for snipmate:

    cd ~/workdir/ && git clone https://github.com/mv/snippets mv-snippets
    cd ~/.vim/    && ln -s ~/workdir/mv-snippets/snipmate-snippets snippets

Inside ~/.vimrc:
    let g:snippets_dir = '~/.vim/snippets'
